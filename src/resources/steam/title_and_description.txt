TITLE

Interface: Century Gothic


DESCRIPTION
-----------

[h1]Mod Description[/h1]
This mod changes all interface fonts to "Century Gothic", the same font used in Stellaris.

[h1]Ironman[/h1]
This mod is Ironman compatible.

[h1]Additional Mods[/h1]
In addition to this mod you can subscribe to the following mods:
[list]
[*] [url=http://steamcommunity.com/sharedfiles/filedetails/?id=734350648]Tooltip: Century Gothic Bold 18[/url]: larger font sizes in tooltips
[*] [url=http://steamcommunity.com/sharedfiles/filedetails/?id=739149045]Interface: Stellaris Colors[/url]: Stellaris color palette
[*] [url=http://steamcommunity.com/sharedfiles/filedetails/?id=754012065]Century Gothic Map Font - Vanilla[/url] by Parasite03: changes the map font to Century Gothic
[/list]
